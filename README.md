Jupyter/PySpark Notebook - Docker Compose Setup

Простой способ запуска Jupyter Notebook с поддержкой PySpark в Docker контейнере без необходимости авторизации.
Предварительные требования:

    Docker (версия 20.10+)
    Docker Compose (версия 2.0+)
    4+ GB свободной оперативной памяти
    2+ GB свободного места на диске

Быстрый старт
1. Клонирование репозитория:

`git clone https://github.com/lukin-vy/BigData_HW.git`

`cd pyspark-jupyter-docker`

2. Запуск контейнера:
`docker-compose up -d`

3. Открыть Jupyter в браузере по адресу : http://localhost:8888

В папке Work находится файл с решением.
